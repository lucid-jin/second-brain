# Raycast 도입기

## 왜 Raycast를 쓰려고 했나

- macOS에서 **파일 검색**을 더 빠르게 하고 싶었음
- Spotlight이 기본 제공이지만 속도와 기능이 부족
- Alfred와 Raycast를 비교한 결과, **무료 범위가 넓고 진입장벽이 낮은** Raycast를 선택

### Alfred vs Raycast 비교 결과

| | Alfred | Raycast |
|---|---|---|
| 워크플로우/확장 | 유료 (Powerpack) | 무료로 Extension 사용 가능 |
| 클립보드 관리 | 유료 | 무료 |
| 스니펫 | 유료 | 무료 |
| AI 기능 | 없음 | Pro에 포함 |
| 결제 방식 | 일회성 구매 | 구독 ($8/월) |

결론: 새로 시작하면 Raycast가 낫다. 무료로 쓸 수 있는 범위가 넓음.

```bash
brew install --cask raycast
```

## 핵심 기능: Extension

Raycast의 진짜 가치는 **Extension Store**. 앱 실행기를 넘어서 개발 도구 허브가 됨.

### 설치한 Extension 목록

#### 개발자 도구
| Extension | 왜 쓰는지 |
|---|---|
| **Kill Process** | 앱 먹통일 때 바로 죽임. Activity Monitor 안 열어도 됨 |
| **Port Manager** | `localhost:3000` 포트 충돌 시 어떤 프로세스가 쓰는지 확인/종료 |
| **Docker** | 컨테이너 시작/중지/로그를 Raycast에서 바로 |
| **SSH Connection Manager** | SSH 접속 목록 관리 |
| **Homebrew** | brew install/uninstall 터미널 안 열고 |
| **JSON Format** | 클립보드의 JSON을 바로 포맷팅 |

#### 생산성
| Extension | 왜 쓰는지 |
|---|---|
| **Window Management** | 창 반으로 나누기, 정렬. Magnet 앱 대체 |
| **Color Picker** | 화면에서 색상 추출 |
| **Translate** | 텍스트 바로 번역 |
| **System Monitor** | CPU/메모리 사용량 확인 |
| **Timer** | 간단한 타이머 |

#### 서비스 연동
| Extension | 왜 쓰는지 |
|---|---|
| **Slack** | 슬랙 메시지 검색/열기 |
| **Notion** | 노션 페이지 검색 |
| **GitHub** | PR, 이슈, 레포 검색 |
| **Linear** | 이슈 관리 |
| **Google Search** | Raycast에서 바로 검색 |
| **Figma** | 파일 검색/열기 |

## 단축키 충돌 문제와 해결

### 문제
- WebStorm에 이미 단축키가 많아서 전역 Hotkey를 지정하면 충돌남
- Hotkey를 많이 만들면 외우기도 힘듦

### 해결: Alias 방식
전역 단축키 대신 **Raycast 안에서 짧은 별칭(Alias)**을 사용.

| 액션 | Alias | 사용법 |
|---|---|---|
| Kill Process | `kp` | Raycast 열고 → `kp` → Enter |
| Clipboard History | `ch` | Raycast 열고 → `ch` → Enter |
| AI Chat | `ai` | Raycast 열고 → `ai` → Enter |
| Port Manager | `pm` | Raycast 열고 → `pm` → Enter |

**전역 단축키는 Raycast 여는 키 하나만** 기억하면 됨. 나머지는 Alias 2글자로 해결.

### 설정 방법
1. Raycast → `Extensions` 입력 → Enter
2. 원하는 명령어 선택
3. **Alias** 칸에 약자 입력

## 설정 동기화 (Pro 없이)

### 문제
- Cloud Sync는 Pro 유료 기능 ($8/월)
- 회사에서 여러 Mac을 쓰는데 설정을 맞추고 싶음

### 해결: iCloud + 셸 스크립트
직접 백업/복원 스크립트를 만들어서 iCloud Drive에 저장.

#### Raycast Script Command로 등록
`~/scripts/raycast-scripts/` 폴더에 스크립트 저장 후:
1. Raycast → `Cmd + ,` (Settings)
2. Extensions → 하단 `+` → **Add Script Directory**
3. `~/scripts/raycast-scripts` 선택

이후 Raycast에서 `Raycast Backup` / `Raycast Restore` 입력하면 실행됨.

#### zshrc 자동 백업
```bash
# ~/.zshrc에 추가
alias raycast-sync="$HOME/scripts/raycast-sync.sh"
$HOME/scripts/raycast-sync.sh backup &>/dev/null &
```
터미널 열 때마다 백그라운드로 자동 백업.

#### 다른 Mac에서 복원
1. Raycast 설치
2. iCloud에서 스크립트 복사
3. `raycast-sync restore` 실행
4. Raycast 재시작
5. GitHub/Notion 등 서비스 인증만 다시

## 느낀 점

- Spotlight 대체를 넘어서 **개발 도구 런처**로 쓸 수 있음
- Extension이 무료인 게 큼 (Alfred는 워크플로우가 유료)
- 단축키 충돌은 Alias로 깔끔하게 해결
- Pro 없이도 수동 동기화로 충분히 여러 Mac에서 쓸 수 있음
