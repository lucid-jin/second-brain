# macOS & IntelliJ 단축키 모음

## macOS 기본

### 기본 조작
| 단축키 | 설명 |
|---|---|
| `Cmd + Space` | Spotlight / Raycast |
| `Cmd + Tab` | 앱 전환 |
| `Cmd + Q` | 앱 종료 |
| `Cmd + W` | 탭/창 닫기 |
| `Cmd + ,` | 앱 설정 |
| `Cmd + H` | 앱 숨기기 |
| `Cmd + M` | 최소화 |
| `Cmd + Opt + H` | 다른 앱 다 숨기기 |

### 텍스트 편집
| 단축키 | 설명 |
|---|---|
| `Cmd + A` | 전체 선택 |
| `Cmd + Z` | 되돌리기 |
| `Cmd + Shift + Z` | 다시 실행 |
| `Opt + ←/→` | 단어 단위 이동 |
| `Cmd + ←/→` | 줄 처음/끝 이동 |
| `Opt + Delete` | 단어 삭제 |
| `Cmd + Delete` | 줄 삭제 |

### Finder
| 단축키 | 설명 |
|---|---|
| `Cmd + Shift + G` | 경로로 이동 |
| `Cmd + Shift + .` | 숨김 파일 보기 |
| `Space` | Quick Look 미리보기 |

### 스크린샷
| 단축키 | 설명 |
|---|---|
| `Cmd + Shift + 3` | 전체 화면 캡처 |
| `Cmd + Shift + 4` | 영역 선택 캡처 |
| `Cmd + Shift + 4 + Space` | 창 캡처 |
| `Cmd + Shift + 5` | 캡처 옵션 패널 |

### 창/데스크톱
| 단축키 | 설명 |
|---|---|
| `Ctrl + ←/→` | 데스크톱 전환 |
| `Ctrl + ↑` | Mission Control |

## Chrome

### 탭
| 단축키 | 설명 |
|---|---|
| `Cmd + T` | 새 탭 |
| `Cmd + Shift + T` | 닫은 탭 복원 |
| `Cmd + W` | 현재 탭 닫기 |
| `Cmd + 1~8` | 해당 번호 탭으로 이동 |
| `Cmd + 9` | 마지막 탭으로 이동 |
| `Cmd + Opt + ←/→` | 이전/다음 탭 이동 |
| `Cmd + Shift + N` | 시크릿 모드 |

### 주소창/검색
| 단축키 | 설명 |
|---|---|
| `Cmd + L` | 주소창 포커스 |
| `Cmd + F` | 페이지 내 검색 |
| `Cmd + Opt + F` | 찾기 입력란 포커스 |

### 페이지
| 단축키 | 설명 |
|---|---|
| `Cmd + R` | 새로고침 |
| `Cmd + Shift + R` | 캐시 무시 새로고침 (강력 새로고침) |
| `Cmd + ←` | 뒤로 가기 |
| `Cmd + →` | 앞으로 가기 |
| `Cmd + +/-` | 확대/축소 |
| `Cmd + 0` | 확대 초기화 |
| `Space` | 페이지 아래로 스크롤 |
| `Shift + Space` | 페이지 위로 스크롤 |

### 개발자 도구 (DevTools)
| 단축키 | 설명 |
|---|---|
| `Cmd + Opt + I` | DevTools 열기/닫기 |
| `Cmd + Opt + J` | Console 바로 열기 |
| `Cmd + Opt + U` | 소스 보기 |
| `Cmd + Shift + C` | 요소 선택 모드 |

### DevTools 패널 이동
| 단축키 | 설명 |
|---|---|
| `Cmd + ]` | 다음 패널로 이동 (Elements → Console → Sources → Network → ...) |
| `Cmd + [` | 이전 패널로 이동 |
| `Cmd + Shift + P` | 명령어 팔레트 (패널명 입력으로 바로 이동) |
| `Cmd + Shift + D` | DevTools 도킹 위치 전환 (하단/우측/분리) |
| `Cmd + Shift + M` | 모바일 반응형 모드 토글 |
| `Esc` | Console 서랍 토글 (다른 패널에서 Console 같이 보기) |

### DevTools Console
| 단축키 | 설명 |
|---|---|
| `Cmd + K` | Console 로그 지우기 |
| `↑/↓` | 이전/다음 명령어 히스토리 |
| `Cmd + F` | Console 내 검색 |

### DevTools Sources (디버깅)
| 단축키 | 설명 |
|---|---|
| `Cmd + P` | 파일 열기 (Go to File) |
| `Cmd + Shift + O` | 함수/심볼 검색 |
| `Cmd + G` | 특정 줄로 이동 |
| `Cmd + B` | 브레이크포인트 토글 |
| `F8` | 실행 재개 (Resume) |
| `F10` | Step Over |
| `F11` | Step Into |
| `Shift + F11` | Step Out |

### DevTools Network
| 단축키 | 설명 |
|---|---|
| `Cmd + F` | 요청 필터/검색 |
| `Cmd + E` | 네트워크 기록 시작/중지 |
| `Cmd + K` | 네트워크 로그 지우기 |

### 북마크/기타
| 단축키 | 설명 |
|---|---|
| `Cmd + D` | 현재 페이지 북마크 |
| `Cmd + Shift + B` | 북마크 바 토글 |
