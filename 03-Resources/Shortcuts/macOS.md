# macOS 단축키

## 기본 조작
| 단축키 | 설명 |
|---|---|
| `Cmd + Space` | Spotlight / Raycast |
| `Cmd + Tab` | 앱 전환 |
| `Cmd + Q` | 앱 종료 |
| `Cmd + W` | 탭/창 닫기 |
| `Cmd + ,` | 앱 설정 |
| `Cmd + H` | 앱 숨기기 |
| `Cmd + M` | 최소화 |
| `Cmd + Opt + H` | 다른 앱 다 숨기기 |

## 텍스트 편집
| 단축키 | 설명 |
|---|---|
| `Cmd + A` | 전체 선택 |
| `Cmd + Z` | 되돌리기 |
| `Cmd + Shift + Z` | 다시 실행 |
| `Opt + ←/→` | 단어 단위 이동 |
| `Cmd + ←/→` | 줄 처음/끝 이동 |
| `Opt + Delete` | 단어 삭제 |
| `Cmd + Delete` | 줄 삭제 |

## Finder
| 단축키 | 설명 |
|---|---|
| `Cmd + Shift + G` | 경로로 이동 |
| `Cmd + Shift + .` | 숨김 파일 보기 |
| `Space` | Quick Look 미리보기 |

## 스크린샷
| 단축키 | 설명 |
|---|---|
| `Cmd + Shift + 3` | 전체 화면 캡처 |
| `Cmd + Shift + 4` | 영역 선택 캡처 |
| `Cmd + Shift + 4 + Space` | 창 캡처 |
| `Cmd + Shift + 5` | 캡처 옵션 패널 |

## 창/데스크톱
| 단축키 | 설명 |
|---|---|
| `Ctrl + ←/→` | 데스크톱 전환 |
| `Ctrl + ↑` | Mission Control |
