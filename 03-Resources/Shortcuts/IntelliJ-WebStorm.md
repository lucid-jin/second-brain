# IntelliJ / WebStorm 단축키 (macOS 키맵)

## 검색
| 단축키 | 설명 |
|---|---|
| `Shift + Shift` | 전체 검색 (Search Everywhere) |
| `Cmd + Shift + F` | 프로젝트 내 텍스트 검색 |
| `Cmd + Shift + O` | 파일 검색 |
| `Cmd + O` | 클래스 검색 |
| `Cmd + Opt + O` | 심볼 검색 |
| `Cmd + E` | 최근 파일 |
| `Cmd + Shift + E` | 최근 수정한 파일 |

## 편집
| 단축키 | 설명 |
|---|---|
| `Cmd + D` | 줄 복제 |
| `Cmd + Delete` | 줄 삭제 |
| `Cmd + /` | 줄 주석 토글 |
| `Cmd + Shift + /` | 블록 주석 토글 |
| `Opt + ↑/↓` | 선택 영역 확장/축소 |
| `Cmd + Shift + ↑/↓` | 줄 이동 |
| `Cmd + Shift + U` | 대소문자 전환 |
| `Ctrl + G` | 다중 커서 (같은 단어 선택) |

## 코드
| 단축키 | 설명 |
|---|---|
| `Cmd + B` | 선언으로 이동 |
| `Cmd + Opt + B` | 구현으로 이동 |
| `Cmd + U` | 상위 클래스/메서드 이동 |
| `Cmd + P` | 파라미터 정보 |
| `Cmd + Shift + Enter` | 문장 완성 |
| `Opt + Enter` | 퀵 픽스 / 제안 |
| `Cmd + Opt + L` | 코드 포맷팅 |
| `Ctrl + Opt + O` | import 정리 |
| `Cmd + Opt + T` | 코드 감싸기 (if, try 등) |

## 리팩토링
| 단축키 | 설명 |
|---|---|
| `Shift + F6` | 이름 변경 |
| `Cmd + Opt + V` | 변수 추출 |
| `Cmd + Opt + M` | 메서드 추출 |
| `Cmd + Opt + C` | 상수 추출 |

## 실행/디버그
| 단축키 | 설명 |
|---|---|
| `Ctrl + R` | 실행 |
| `Ctrl + D` | 디버그 |
| `Cmd + F8` | 브레이크포인트 토글 |
| `F8` | Step Over |
| `F7` | Step Into |
| `Shift + F8` | Step Out |
| `Cmd + F2` | 실행 중지 |

## 탭/창
| 단축키 | 설명 |
|---|---|
| `Cmd + Shift + [/]` | 탭 전환 |
| `Cmd + 1` | Project 패널 토글 |
| `Cmd + 4` | Run 패널 토글 |
| `Cmd + 5` | Debug 패널 토글 |
| `Cmd + 9` | Git 패널 토글 |
| `Cmd + Shift + F12` | 에디터 최대화 |

## Git
| 단축키 | 설명 |
|---|---|
| `Cmd + K` | 커밋 |
| `Cmd + Shift + K` | 푸시 |
| `Cmd + Opt + Z` | 변경 취소 (Revert) |
